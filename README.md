一个慕课网的实例教程-基于HTML5的WEB阅读器，后期本人做了一些优化

## Done

- 页面架构
- 触摸交互
- 字体和页面背景设置交互
- 页面数据获取

## 一些优化点

- 取消多余的字体和背景设置变量，直接用style中background-color和color属性来完成功能
- 白天/黑夜模式切换过程会存储上一个背景，即点击按钮界面回到最近一次的白天/黑夜背景
